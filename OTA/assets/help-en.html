OTA Upgrade Help Document

OTA upgrade feature upgrade online and local upgrade, please refer to the following method of operation:

1.Online Upgrade:

a.Open WIFI network (Download consuming network traffic, it is recommended to use WIFI);

b.Click on the "Upgrade" button to get the latest upgrade package;

c.Upgrade package after search, click on the "download" button to download the upgrade package to a local disk;

d.Click the "Back" button to download the upgrade package can cancel the current download;

e.After the download is complete, click the "Upgrade Now" button and follow the prompts to upgrade;

2.Local Upgrades:

a.First make sure the local disk the update.zip format OTA upgrade package, or use the online upgrade feature to obtain the upgrade package;

b.Click on the "Local Upgrade" button automatically from the local disk, the update.zip format the OTA upgrade package, and then click on the "OK" button to automatically enter the Recovery upgrade;

3.Setting:

a.Check the automatic check update automatically access the server from time to time whether there is an updated version, the user is prompted, no not prompted;

b.Check the automatic checking for updates, you can customize the polling period;

b.Uncheck automatic checking for updates, no automatic access to the server later, you need the user to manually click "Online Upgrade" button to query;



Precautions:

1.Before the upgrade, make sure the battery is more than 90%, Do not power off;

2.Download online upgrade, do not power off or forced shutdown the download process system shutdown or restart, the system will automatically stop the current download, system startup users re-download the upgrade package;

3.The upgrade process, do not power off or forced shutdown after the upgrade is complete, the device will automatically restart;

4.Upgrade process in the event of an accident and can not enter the system, by the way of the SD card reset the system to reset the system;

5.SD card upgrade still can not enter the system, please contact the official after-sales service.
